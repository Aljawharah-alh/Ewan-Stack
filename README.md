# EWAN
**A Full-Stack Event Discovery & Booking Platform**

## Overview
EWAN is a comprehensive Full-Stack Web Application designed to showcase and manage vibrant events across different Saudi cities. The platform provides a cinematic user experience with a robust backend to handle real-time bookings and user data management.

## Key Features
* Full-Stack Integration: A seamless connection between a dynamic frontend and a powerful Django backend.
* Saudi Event Discovery: High-end visual interface for exploring activities across the Kingdom's cities.
* Integrated Booking System: A complete logic for users to reserve tickets, manage quantities, and track their bookings.
* Secure User Authentication: Private accounts for users to manage their personal event schedules and history.
* Responsive Cinematic UI: Optimized experience for all devices using modern CSS and Glassmorphism effects.

## Scientific Implementation Details
* CRUD Operations: Implemented full Create, Read, Update, and Delete logic for the booking system.
* Session Management: Utilized secure server-side sessions to maintain user authentication and state.
* Relational Database Design: Structured complex relationships between City, Event, and User models.
* Template Inheritance: Engineered a modular UI system using Django’s template engine for scalable code.

## Installation & Setup
1. Clone the repo: git clone https://github.com/your-username/EWAN-Platform.git
2. Install dependencies: pip install -r requirements.txt
3. Run migrations: python manage.py migrate
4. Start server: python manage.py runserver
